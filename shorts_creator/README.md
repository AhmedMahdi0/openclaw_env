# 🎬 AI Video Shorts Creator

تحويل الصور القديمة إلى فيديوهات قصيرة مذهلة باستخدام الذكاء الاصطناعي!  
هذا المشروع يأخذ صورك الشخصية ويحولها إلى فيديوهات متحركة، ثم يجمعها في شورت جاهز للنشر على YouTube Shorts، TikTok، أو Instagram Reels.

## ✨ المميزات

- **🖼️ معالجة صور ذكية**: تحسين الجودة، اقتصاص، وتجهيز الصور للذكاء الاصطناعي
- **🤖 توليد فيديو بالذكاء الاصطناعي**: تحويل الصور إلى فيديوهات متحركة باستخدام RunwayML أو Pika Labs
- **🎵 تجميع تلقائي**: دمج الفيديوهات مع موسيقى خلفية وعناوين
- **📱 متوافق مع Shorts**: حجم عملي (1080x1920) ومدة مثالية (15-60 ثانية)
- **🔧 سهل الاستخدام**: واجهة سطر أوامر بسيطة مع إرشادات خطوة بخطوة

## 📁 هيكل المشروع

```
shorts_creator/
├── input_images/          # ضع صورك هنا (JPG, PNG, إلخ)
├── enhanced_images/       # الصور المحسنة تلقائياً
├── generated_videos/      # الفيديوهات المولدة بالذكاء الاصطناعي
├── final_videos/          # الشورتات النهائية الجاهزة
├── audio/                 # موسيقى خلفية (MP3, WAV) - اختياري
├── image_processor.py     # معالجة الصور
├── video_generator.py     # توليد الفيديو بالذكاء الاصطناعي
├── video_assembler.py     # تجميع الفيديو النهائي
├── run_pipeline.sh        # سكربت التشغيل الكامل
├── requirements.txt       # متطلبات بايثون
├── .env.example          # مثال لملف الإعدادات
└── README.md             # هذا الملف
```

## 🚀 البدء السريع

### 1. التثبيت

```bash
# انسخ المشروع
git clone <repository-url>
cd shorts_creator

# أنشئ بيئة افتراضية (اختياري)
python3 -m venv venv
source venv/bin/activate  # على Linux/Mac
# أو
venv\Scripts\activate     # على Windows

# ثبت المتطلبات
pip install -r requirements.txt
```

### 2. الإعداد

```bash
# انسخ ملف الإعدادات
cp .env.example .env

# عدل ملف .env بإعداداتك (اختياري)
# أضف مفاتيح API إذا كان لديك
```

### 3. إضافة الصور

```bash
# ضع صورك في مجلد input_images
mkdir -p input_images
# انسخ صورك إلى هذا المجلد
```

### 4. التشغيل

```bash
# اجعل سكربت التشغيل قابلاً للتنفيذ
chmod +x run_pipeline.sh

# تشغيل البايبلاين الكامل
./run_pipeline.sh

# أو تشغيل الخطوات يدوياً
python3 image_processor.py
python3 video_generator.py
python3 video_assembler.py
```

## 📋 خطوات العملية

### **الخطوة 1: معالجة الصور**
```bash
python3 image_processor.py
```
- يبحث عن الصور في `input_images/`
- يحسن الجودة، يغير الحجم للعمودي (1080x1920)
- يحفظ الصور المحسنة في `enhanced_images/`

### **الخطوة 2: توليد الفيديو**
```bash
python3 video_generator.py
```
- يختار بين:
  1. **Mock Generation**: تجربة بدون API (للتجربة)
  2. **RunwayML API**: يحتاج مفتاح API
  3. **Pika Labs API**: يحتاج مفتاح API
- يحول كل صورة إلى فيديو قصير (3-5 ثواني)
- يحفظ في `generated_videos/`

### **الخطوة 3: التجميع النهائي**
```bash
python3 video_assembler.py
```
- يجمع الفيديوهات معاً
- يضيف عناوين وتأثيرات
- يضيف موسيقى خلفية من `audio/` (إذا موجودة)
- يخلق فيديو نهائي في `final_videos/`

## 🔑 APIs المدعومة

### **RunwayML** (موصى به)
- جودة عالية، حركة واقعية
- [سجل للحصول على API key](https://runwayml.com)

### **Pika Labs**
- سهل الاستخدام، نتائج جيدة
- [سجل للحصول على API key](https://pika.art)

### **بدون API (Mock Mode)**
- للتجربة والاختبار
- ينشئ فيديوهات تجريبية بسيطة

## ⚙️ الإعدادات المتقدمة

### **تعديل ملف .env:**
```env
# أبعاد الفيديو
VIDEO_WIDTH=1080
VIDEO_HEIGHT=1920
VIDEO_FPS=30
VIDEO_DURATION_PER_IMAGE=3

# APIs (اختياري)
RUNWAYML_API_KEY=your_key_here
PIKA_API_KEY=your_key_here
OPENAI_API_KEY=your_key_here  # للنصوص
```

### **إضافة موسيقى خلفية:**
- ضع ملفات MP3 أو WAV في مجلد `audio/`
- البرنامج سيختار عشوائياً ويضيفها تلقائياً

## 💡 نصائح للحصول على أفضل نتائج

### **للصور:**
1. **جودة عالية**: صور واضحة وذات إضاءة جيدة
2. **تركيز على الوجه**: الصور الشخصية تعمل بشكل أفضل
3. **خلفية بسيطة**: تساعد الذكاء الاصطناعي على التركيز
4. **5-10 صور**: العدد المثالي لشورت واحد

### **للفيديو النهائي:**
1. **مدة 15-45 ثانية**: مثالي لـ Shorts
2. **موسيقى هادئة**: لا تطغى على الصوت
3. **عناوين قصيرة**: واضحة وسهلة القراءة
4. **حركة خفيفة**: أكثر واقعية وطبيعية

## 🚀 النشر

### **على YouTube Shorts:**
1. افتح تطبيق YouTube
2. انقر على "+" → Create → Create Short
3. اختر الفيديو من `final_videos/`
4. أضف وصف وجزء من القصة
5. انشر!

### **على TikTok:**
1. افتح تطبيق TikTok
2. انقر على "+" لإنشاء فيديو
3. اختر الفيديو من `final_videos/`
4. أضف تأثيرات وموسيقى
5. انشر!

### **على Instagram Reels:**
1. افتح تطبيق Instagram
2. انزل لليسار لفتح الكاميرا
3. اختر Reels
4. اختر الفيديو من `final_videos/`
5. أضف تأثيرات وموسيقى
6. انشر!
