# ⚡ البدء السريع

## في 5 دقائق فقط، أنشئ أول فيديو لك!

### الخطوة 1: التثبيت (دقيقة واحدة)
```bash
# تأكد من وجود بايثون
python3 --version

# ثبت المتطلبات
pip install -r requirements.txt
```

### الخطوة 2: أضف صورك (دقيقة واحدة)
```bash
# أنشئ مجلد الصور إذا لم يكن موجوداً
mkdir -p input_images

# انسخ 3-5 صورك إلى المجلد
# (استخدم أي طريقة تناسبك: نسخ، سحب، إلخ)
```

### الخطوة 3: التشغيل (3 دقائق)
```bash
# الطريقة الأسهل: استخدم سكربت التشغيل الكامل
chmod +x run_pipeline.sh
./run_pipeline.sh

# أو تشغيل الخطوات يدوياً:
# 1. معالجة الصور
python3 image_processor.py

# 2. توليد الفيديو (اختار 1 للمحاكاة)
python3 video_generator.py

# 3. التجميع النهائي
python3 video_assembler.py
```

### الخطوة 4: ابحث عن الفيديو النهائي
```bash
# الفيديو النهائي سيكون في:
ls -la final_videos/

# أحدث فيديو:
ls -t final_videos/*.mp4 | head -1
```

## 🎯 مثال عملي كامل

### 1. جهز 3 صور:
- صورة طفولة
- صورة من رحلة
- صورة حديثة

### 2. تشغيل الأوامر:
```bash
# 1. معالجة الصور
echo "معالجة الصور..."
python3 image_processor.py

# 2. توليد الفيديو (اختر 1 للمحاكاة)
echo "توليد الفيديوهات..."
echo "1" | python3 video_generator.py

# 3. التجميع النهائي
echo "تجميع الفيديو النهائي..."
echo "1" | python3 video_assembler.py
```

### 3. النتيجة:
- فيديو مدته 30-45 ثانية
- حجم 1080x1920 (عمودي)
- جاهز للنشر على Shorts

## 🔧 إعدادات سريعة

### لتحسين النتائج:
1. **أضف موسيقى** إلى مجلد `audio/`
2. **عدل ملف `.env`** لإضافة API keys حقيقية
3. **استخدم صور عالية الجودة**

### حجم الصور المثالي:
- دقة عالية (1080x1920 أو أكبر)
- إضاءة جيدة
- تركيز على الوجه (للصور الشخصية)

## 🚀 الانتقال للإنتاج

### للحصول على نتائج أفضل:
1. **سجل في RunwayML** واحصل على API key
2. **أضف المفتاح** إلى ملف `.env`
3. **استخدم خدمة runwayml** بدلاً من mock

```bash
# نسخ ملف الإعدادات
cp .env.example .env

# تعديل .env بإضافة:
RUNWAYML_API_KEY=مفتاحك_هنا
```

## ❓ مساعدة سريعة

### إذا واجهت مشكلة:
```bash
# اختبار النظام
python3 test_pipeline.py

# عرض المساعدة
python3 image_processor.py --help
```

### الأخطاء الشائعة:
1. **لا توجد صور**: تأكد من وجودها في `input_images/`
2. **moviepy غير مثبت**: `pip install moviepy`
3. **صلاحيات**: `chmod +x *.py`

4. **اختر الفيديو** وانشر!
